Here get auth Menu that have some require property for next steps

| Property    | Description                                                                                                                                                                                                                           |
| ----------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `label`     | This label is Display Where we show list of page                                                                                                                                                                                      |
| `template`  | This is template name, and template name is used to Mapping template and Render match template.                                                                                                                                       |
| `url`       | This is page url                                                                                                                                                                                                                      |
| `id`        | id is used to uniquely identify menu in all menu array                                                                                                                                                                                |
| `caches`    | caches is used to store caches of api json data                                                                                                                                                                                       |
| `nextPage`  | This is next page url. After Admin form submit success fully then we can redirect user to next page using                                                                                                                             |
| `param`     | if we want to send some data from one to another Admin pages after form submit successful then param parameter will help them. We got name of form filed from param where we can collect info using name and pass into another pages. |
| `prevParam` | It have data of that is pass by previous pages.                                                                                                                                                                                       |

Here we we call option api that we get endpoint `/noauth/api/login-via-email/` from above json and call option api
